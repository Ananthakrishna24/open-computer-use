# Benchmarks

The guide requires phase 1 to publish a table before claiming the browser vertical slice is a
success. Track the same task suite against:

- `open-computer-use` with a small text-first model.
- browser-use in text mode.
- A screenshot-loop computer-use baseline.

Metrics:

- `tokens/step`
- `$ / task`
- `success %`

The target is at most one fifth of the screenshot loop's tokens at equal or better success.
